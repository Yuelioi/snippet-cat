## 介绍

Snippet Cat 是一款 VSCODE 拓展,用于管理代码片段.

## 功能

- 片段管理,无限层级(文件/文件夹) 
- 两种视图:支持树状 所有文件平铺
- 支持同步(webdav/github/gitee)
- 只支持文本类型同步(为了加快同步速度)
